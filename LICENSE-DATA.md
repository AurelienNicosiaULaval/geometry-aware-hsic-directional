# Data and generated-output license notes

This repository contains software, manuscript sources, generated simulation
outputs and a local Noshiro data export.

The R code and supporting software scripts are licensed under the MIT License
in `LICENSE`.

Generated CSV files, figures, tables and PDFs in `output/final/` are provided
as replication materials for the manuscript. Users should cite the manuscript
and the archived repository DOI when reusing these generated materials.

The Noshiro data file `data/noshiro.csv` is a local CSV export of
`ridgetorus::earthquakes`. The source R package `ridgetorus` version 1.0.3
declares GPL-3 in its package `DESCRIPTION`. Provenance notes and original
references are recorded in `data/noshiro_source.txt`. No additional license is
asserted here for third-party source data beyond the source package and
original-data terms.

Before external redistribution in another archive or journal supplement,
verify that the Noshiro data reuse terms are compatible with the intended
distribution route.
