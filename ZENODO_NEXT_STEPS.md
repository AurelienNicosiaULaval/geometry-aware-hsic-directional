To mint a Zenodo DOI:

1. Log in to Zenodo with the GitHub account that owns the repository.
2. Open the Zenodo GitHub integration page.
3. Click "Sync now" if the repository list is not current.
4. Enable `AurelienNicosiaULaval/geometry-aware-hsic-directional`.
5. Create a new GitHub release after the repository has been enabled in Zenodo.
6. Wait for Zenodo to archive the release and assign a DOI.
7. Add the Zenodo DOI to `README.md`, `main.tex`, `CITATION.cff` and the GitHub release notes.

Items to verify before publishing the Zenodo record:

- Add an explicit repository license if open reuse terms should be defined.
- Add ORCID only if the verified ORCID should appear in the public metadata.
- Check whether the archive should be linked as software or as supplementary manuscript material.
- Confirm that the archived version is the one intended for CSDA submission.
